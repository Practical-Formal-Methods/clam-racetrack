Model.getPWLObj()

getPWLObj ( var )

Retrieve the piecewise-linear objective function for a variable. The function returns a list of tuples, where each provides the and coordinates for the points that define the piecewise-linear objective function.

Refer to this discussion for additional information on what the values in and mean.

Arguments:

var: A Var object that gives the variable whose objective function is being retrieved.

Return value:

The points that define the piecewise-linear objective function.

Example usage:

  > print(model.getPWLObj(var))
  [(1, 1), (2, 2), (3, 4)]
