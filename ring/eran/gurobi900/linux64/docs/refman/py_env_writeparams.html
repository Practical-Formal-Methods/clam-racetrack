Env.writeParams()

writeParams ( filename )

Write all modified parameters to a file. The file is written in PRM format.

Example usage:

  env.setParam("Heu*", 0.5)
  env.writeParams("params.prm")  # file will contain changed parameter
  system("cat params.prm")
