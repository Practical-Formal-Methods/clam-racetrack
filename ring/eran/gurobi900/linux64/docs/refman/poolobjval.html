PoolObjVal

Type: double
Modifiable: No

This attribute is used to query the objective value of the -th solution stored in the pool of feasible solutions found so far for the problem. You set using the SolutionNumber parameter.

The number of stored solutions can be queried using the SolCount attribute.

For examples of how to query or modify attributes, refer to our Attribute Examples.
