Model.getVarByName()

getVarByName ( name )

Retrieve a variable from its name. If multiple variables have the same name, this method chooses one arbitrarily.

Arguments:

name: Name of desired variable.

Return value:

Variable with the specified name.

Example usage:

  x0 = model.getVarByName("x0")
