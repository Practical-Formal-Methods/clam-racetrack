Model.getA()

getA ( )

Query the linear constraint matrix of the model. You'll need to have scipy installed for this function to work.

Arguments:

Return value:

The matrix as a scipy.sparse matrix in CSR format.

Example usage:

  A = model.getA()
  print(A.toarray())
